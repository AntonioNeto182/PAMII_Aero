Trabalho interdisciplinar 4 Bimestre, feito enquanto eu cursava DS (Desenvolvimento de Sistemas) em 2025 na ETEC de Registro-SP.
O projeto consiste em um aplicativo de reservas de passagens aéreas nacionais, com suporte a CRUD, criação e login de contas, não foi pensado para ser seguro então não usa nenhum tipo de criptografia ou de medidas de mitigação a ataques. 
O aplicativo foi desenvolvido utilizando React Native e executado pelo Expo. O backend utiliza PHP para comunicação com um banco de dados SQL hospedado em servidor próprio.
O backend (API em PHP + MySQL) roda localmente e não está incluído no repositório por conter credenciais sensíveis.
