Trabalho interdisciplinar do 4º bimestre, desenvolvido durante o curso de Desenvolvimento de Sistemas (ETEC de Registro – 2025).
O projeto consiste em um aplicativo para reservas de passagens aéreas nacionais, incluindo:

CRUD de voos, reservas e usuários

criação de contas e autenticação

integração entre frontend e backend

O aplicativo foi desenvolvido em React Native (Expo).
O backend utiliza PHP + MySQL, com comunicação via API própria.
